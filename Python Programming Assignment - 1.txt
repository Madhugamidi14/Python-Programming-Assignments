4. Write a Python program to swap two variables?

Ans : 

a = 10
b = 11

print('Before swap:\na = {} and b = {}'.format(a, b))

temp = a
a = b
b = temp

print('\nAfter swap:\na = {} and b = {}'.format(a, b))
